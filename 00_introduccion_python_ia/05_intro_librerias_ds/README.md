# Módulo 0.5: Introducción a Librerías de Data Science

Python por sí solo es lento. Las librerías de Data Science usan C/C++ por debajo para ser rápidas.

## 1. NumPy (Numerical Python)

La base de todo.
*   **Arrays:** Listas supervitaminadas. Permiten operaciones vectorizadas (sumar dos listas de 1 millón de números en microsegundos).
*   **Uso en IA:** Los Embeddings son arrays de NumPy.

## 2. Matplotlib (Visualización)

Es vital ver qué está pasando.
*   Graficar la pérdida (loss) durante el entrenamiento.
*   Visualizar clusters de datos.
