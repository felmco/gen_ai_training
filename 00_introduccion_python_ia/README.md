# Fase 0: Introducción a Python para Inteligencia Artificial

Esta fase está diseñada para desarrolladores que necesitan refrescar o aprender Python con un enfoque 100% orientado a Datos e IA. No perderemos tiempo en conceptos que no se usan en este campo.

## Objetivos
1.  Entender cómo los conceptos básicos de programación mapean a conceptos de IA (Bucles -> Entrenamiento, Variables -> Hiperparámetros).
2.  Dominar las estructuras de datos necesarias para manipular Datasets y JSONs.
3.  Aprender la Programación Orientada a Objetos (POO) necesaria para usar frameworks como LangChain o PyTorch.
4.  Introducción a la eficiencia numérica con NumPy.

## Módulos
*   **0.1 Fundamentos Básicos:** El ciclo de entrenamiento simulado.
*   **0.2 Estructuras de Datos:** Manipulación de historiales de chat y vocabularios.
*   **0.3 Funciones:** Matemáticas de redes neuronales y tipado.
*   **0.4 POO:** Construcción de Agentes.
*   **0.5 Librerías DS:** Por qué Python necesita C++ (NumPy).
