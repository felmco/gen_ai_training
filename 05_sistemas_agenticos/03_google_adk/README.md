# Módulo 5.3: Google Agent Development Kit (ADK)

El ADK de Google representa un enfoque de ingeniería de software riguroso para la construcción de agentes, en contraste con el enfoque más experimental de otros frameworks.

## 1. Ingeniería vs. Scripting

Mientras que LangChain nació del scripting rápido, ADK nace de la necesidad empresarial de:
*   **Tipado Estricto:** Interfaces claras entre agentes.
*   **Escalabilidad:** Despliegue en Google Cloud Vertex AI.
*   **Multimodalidad Nativa:** Audio y Video en tiempo real.
